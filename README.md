# HospiPlus 🏥

Sistema web de gestión hospitalaria desarrollado con ASP.NET MVC 5, enfocado en la administración de pacientes y procesos médicos.

## Tecnologías utilizadas
- C#
- ASP.NET MVC 5
- .NET Framework
- SQL Server
- Entity Framework
- Bootstrap

## Funcionalidades principales
- Gestión de pacientes y médicos
- Programación y control de citas médicas (con límite por día)
- Registro de consultas y recetas
- Gestión de exámenes médicos
- Sistema de autenticación de usuarios
- Generación de reportes (pacientes, consultas, recetas y exámenes)

## Descripción
HospiPlus permite digitalizar y optimizar los procesos de una clínica, facilitando la gestión de información médica y mejorando el control de citas, consultas y expedientes de pacientes.
